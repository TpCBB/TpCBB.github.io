you-get简单使用

发表于 2020-07-29 | 分类于 tech ， [object Object]

you-get 环境

安装python3 python3.8.5 自带pip
python各个版本
安装you-get依赖 ffmpeg
ffmpeg版本
安装you-get pip3 install you-get

you-get 使用

$ you-get -i url
you-get -c [cookies.txt/cookies.sqlite] (本地cookies地址) (视频格式 通过 -i 看出) (视频链接)
例子 $ you-get -c F:\test2\cookies.txt --format=flv360 https://www.bilibili.com/bangumi/play/ep330408

暂时只用到这些 下b站会员视频 emmm
tips. 链接不用加单引号
